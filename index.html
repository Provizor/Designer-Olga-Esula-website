    Формулировка задачи

  • 2

    Заказ проекта

    Отправить данные
    Подготовьте всё необходимое для работы (брэнд-бук, иллюстрации, фото, текст для каждого блока) и отправьте удобным вам способом. Это и будет заявкой на проект.
    В ответ вам будет прислано техническое задание для согласования, если оно уже есть - присылайте с брифом!

    Внести предоплату
    По техническому заданию создается структура макета, на основании которой определяется стоимость проекта и сроки работы. После согласования вноситься предоплата (20% от стоимости.).

    Приёмка работы

    Оплатить
